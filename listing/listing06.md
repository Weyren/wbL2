Что выведет программа? Объяснить вывод программы. Рассказать про внутреннее устройство слайсов и что происходит при передачи их в качестве аргументов функции.

```go
package main

import (
	"fmt"
)

func main() {
	var s = []string{"1", "2", "3"}
	modifySlice(s)
	fmt.Println(s)
}

func modifySlice(i []string) {
	i[0] = "3"
	i = append(i, "4")
	i[1] = "5"
	i = append(i, "6")
}
```

Ответ:
```
Вывод: [3 2 3]
В функцию modifySlice попадает копия изначального слайса.
Поскольку изначальный слайс и его копия ссылаются на один и тот же массив,
после i[0] = "3" слайс будет выглядеть так: [3 2 3]
Далее происходит добавление нового элемента в слайс, и поскольку емкость переполняется,
append возвращает слайс, который указывает на новую выделенную область памяти
Дальнейшие операции над слайсом уже не будут изменять массив изначального слайса

Внутреннее устройство слайса:

type slice struct {
	array unsafe.Pointer -указатель на массив 
	len   int -длина слайса(количество элементов)
	cap   int -емкость слайса(размер базового массива)
}

При передаче слайса в функцию он передается по значению.
В функции создается копия слайса, но это приводит к неявному поведению:
изменение значений по индексу в копии влияет на элементы базового слайса,
а добавление новых - нет

```
